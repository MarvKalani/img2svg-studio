# Demo asset

`marv-kalani-logo.jpg` was provided by the project owner on 20 July 2026 for the product demo and
video. It is a first-party visual asset, not a geometric ground-truth fixture.

`topography-island.png` was generated specifically for this project with OpenAI image generation
on 21 July 2026. It depicts invented geography and contains no external map or elevation data. Its
dense contour lines and flat elevation bands make it a reproducible complexity benchmark, not a
semantic terrain dataset.
